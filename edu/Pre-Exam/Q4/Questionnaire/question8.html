Question 8

You have an Azure virtual machine named VM1 that runs Windows Server.

The operating system on VM1 fails to start due to a disk error.

You need to resolve the error.

Which four commands should you run in sequence in Azure Cloud Shell? To answer, move the appropriate commands from the list of commands to the answer area and arrange them in the correct order.

az vm disk
az vm repair restore
az vm repair create
az extension add
az vm repair run
Step 1
Step 2
Step 3
Step 4

Explanation:

Run the following commands in sequence:

  1. Run az extension add to add the required extension.
  2. Run az vm repair create to create a repair configuration.
  3. Run az vm repair run to perform the repair actions.
  4. Run az vm repair restore to restore the VM.

Step 1: az extension add
