Question 01

You have 200 Azure virtual machines.

You create a recovery plan in Azure Site Recovery to fail over all the virtual machines to an Azure region. The plan has three manual actions.

You need to replace one of the manual actions with an automated process.

What should you use?





Explanation:

Automate tasks in recovery plans. Recovering large applications can be a complex task. Manual steps make the process prone to error, and the person running the failover might not be aware of all app intricacies. You can use a recovery plan to impose order and automate the actions needed at each step, using Azure Automation runbooks for failover to Azure, or scripts.
