Question 20

You have five Azure virtual machines.
You need to collect performance data and Windows Event logs from the virtual machines. The data collected must be sent to an Azure Storage account.
What should you install on the virtual machines?





Explanation:

Use Azure Diagnostics extension if you need to:
Send data to Azure Storage for archiving or to analyze it with tools such as Azure Storage Explorer. Diagnostics extension: Sends data to Azure Monitor Metrics (Windows only), Azure Event Hubs, and Azure Storage. This is not consolidated yet. So, Diagnostics extension is a legacy extension that will be replaced completely by Azure Monitor agent in the near future but right now is the right answer. Moreover, the emphasis is on the "Send data to Azure Storage" and the Diagnostics extension does exactly that.
