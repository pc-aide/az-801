Question 16

You have an Azure virtual machine named VM1. Crash dumps for a process named Process1 are enabled for VM1. When process1.exe on VM1 crashes, a technician must access the memory dump files on the virtual machine. The technician must be prevented from accessing the virtual machine. To what should you provide the technician access?




Explanation:

The Azure Diagnostics extension (Agent) is used to send data to Azure Storage for archiving or to analyze it with tools such as Azure Storage Explorer. By providing the technician access to an Azure Blob Storage container, you can ensure they can retrieve the memory dump files without having direct access to the virtual machine itself.

References:

  • Azure Diagnostics Extension Overview
  • Supported Data Types for Diagnostics Extension

Correct Answer:
C: an Azure Blob Storage container
