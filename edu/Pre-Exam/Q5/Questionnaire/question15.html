Question 15

You have an Azure virtual machine named VM1 that has the Web Server (IIS) server role installed. VM1 hosts a critical line-of-business (LOB) application. After the security team at your company deploys a new security baseline to VM1, users begin reporting that the application is unresponsive. You suspect that the security baseline has caused networking issues. You need to perform a network trace on VM1. What should you do?




Explanation:

The Azure Performance Diagnostics extension is a powerful tool for diagnosing and identifying performance issues with Azure VMs. It allows you to gather a wide range of performance data, including network traces, which can be invaluable in situations where you suspect networking issues. Configuring Performance Diagnostics on VM1 through the Azure portal will enable you to collect the necessary data to understand what's causing the application to become unresponsive.

References:

  • Performance Diagnostics for Azure VMs

Correct Answer:
D: From the Azure portal, configure the Performance diagnostics settings for VM1
