Question 19

You have an Azure virtual machine named VM1 that runs Windows Server.

The operating system on VM1 fails to fully initialize its network stack, and you cannot establish a network connection.

You need to establish an interactive shell session.

What should you use?




Explanation:

The Serial Console provides direct access to the VM's console without relying on a functional network stack, which is essential in this scenario where the network stack fails to initialize.

References:

  • Serial Console Overview

Correct Answer:
B: Serial console
