Question 11

You have an Azure virtual machine named VM1 that runs Windows Server.

You plan to deploy a new line-of-business (LOB) application to VM1.

You need to prevent the application from creating child processes.

What should you configure on VM1?




Explanation:

To prevent an application from creating child processes, you can use Exploit Protection within Microsoft Defender. This setting is part of the Attack Surface Reduction (ASR) rules which help minimize the security risk by controlling certain behaviors of applications, such as preventing child process creation.

In the Group Policy Editor, the path to configure this setting is:

Configuration\Administrative Templates\Windows Components\Microsoft Defender Antivirus\Microsoft Defender Exploit Guard\Attack Surface Reduction\Exclude files and paths from Attack Surface Reduction Rules

References:

  • Customize Exploit Protection in Microsoft Defender

Correct Answer:
D: Exploit protection
