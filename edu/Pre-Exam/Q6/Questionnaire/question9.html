Question 9

You have an Azure virtual machine named VM1 that runs Windows Server.

When you attempt to install the Azure Performance Diagnostics extension on VM1, the installation fails.

You need to identify the cause of the installation failure.

What are two possible ways to achieve the goal? Each correct answer presents a complete solution.




Explanation:

The `MonitoringAgent.log` file is related to the Azure Monitoring Agent, which is not referenced in this scenario, so option A is not a suitable answer. Instead, verifying the `WaAppAgent.log` file and viewing the activity log from the Azure portal are appropriate methods for troubleshooting the installation failure.

References:

  • Troubleshoot Azure Performance Diagnostics Extension Installation
  • Monitor and Analyze Virtual Machines

Correct Answers:
B: Sign in to VM1 and verify the WaAppAgent.log file.
D: From the Azure portal, view the activity log for VM1.
