Question 8

You have five Azure virtual machines. You have a dedicated Azure Storage account to collect performance data.

You need to send the collected data directly to the Azure Storage account.

What should you install on the virtual machines?





Explanation:

At the time of writing, the Monitoring agent does not support Azure Storage, which is a requirement in the question. Therefore, the answer should be the Diagnostics Extension, even though the Monitoring agent will replace the Diagnostics Extension. This is supported by the "Compare to legacy agents" overview in the following sources.

References:

  • Azure Diagnostics Extension Overview
  • Supported Services and Features
  • Consolidating Legacy Agents

Correct Answer:
E: the Azure Diagnostics extension
