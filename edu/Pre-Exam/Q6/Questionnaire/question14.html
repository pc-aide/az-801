Question 14

You have three servers named Server1, Server2, and Server3 that run Windows Server and have the Hyper-V server role installed. Server1 hosts an Azure Migrate appliance named Migrate1.

You plan to migrate virtual machines to Azure.

You need to ensure that any new virtual machines created on Server1, Server2, and Server3 are available in Azure Migrate.

What should you do?





Explanation:

A discovery source specifies the Hyper-V hosts that Azure Migrate will discover and assess for migration. When you add a discovery source, Azure Migrate automatically detects and adds any new virtual machines that are created on the specified Hyper-V hosts.

References:

  • Azure Migrate: Discover Hyper-V VMs

Correct Answer:
D: On Migrate1, add a discovery source.
