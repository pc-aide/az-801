Question 01

You have a server named Server1 that runs Windows Server. On Server1, you create a Data Collector Set named CollectorSet1 based on the Basic template. You need to configure CollectorSet1 to meet the following requirements:

  • Older performance counter logs must be overwritten by new ones.
  • Performance counter logging must stop if there is less than 500 MB of free disk space.

What should you configure for each requirement? To answer, select the appropriate options in the answer area.


Explanation:

To overwrite older performance counter logs by new ones, select the "Overwrite" option for the "File name" setting in the data collector set properties. You can access the properties by right-clicking on the data collector set in Performance Monitor and selecting "Properties". In the "Data Collector Set Properties" dialog box, click the "Log Files" tab, and then select "Overwrite" for the "File name" setting.
