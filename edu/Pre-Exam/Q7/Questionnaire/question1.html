Question 01

Your on-premises network contains an Active Directory Domain Services (AD DS) domain. The domain contains 10 servers that run Windows Server.

You have an Azure subscription. The subscription contains 10 virtual machines that run Windows Server.

You need to install the Azure Monitor Agent on the Azure virtual machines and the on-premises servers.

What should you use? To answer, select the appropriate options in the answer area.


Explanation:

Virtual Machine Extensions are designed for Azure VMs and provide a way to install and manage agents like the Azure Monitor Agent. For on-premises servers, use the Windows Installer (MSI) package.

References:

  • Virtual Machine Extensions Overview
  • Windows Installer Portal

Correct Answers:
For the Azure virtual machines: Virtual machine extensions
For the on-premises servers: The Windows installer (MSI) package
