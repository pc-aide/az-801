Question 11

You have a failover cluster named Cluster1 that hosts an application named App1.

The General tab in App1 Properties is shown in the General exhibit.

The Failover tab in App1 Properties is shown in the Failover exhibit.

You need to ensure that when you start Server1, App1 continues to run on Server2.

Solution: You increase Maximum failures in the specified period for the App1 cluster role. Does this meet the goal?

Explanation:

The Maximum failures setting is used to determine when the cluster considers a node as offline. It does not affect whether a cluster will fail back when a node comes online. This setting does not ensure that App1 continues to run on Server2 if Server1 is started.

Correct Answer:

  • No
