Question 16

Your network contains an Active Directory Domain Services (AD DS) domain. The domain contains two servers named Server1 and Server2 that run Windows Server.

You need to ensure that you can use the Computer Management console to manage Server2. The solution must use the principle of least privilege.

Which two Windows Defender Firewall with Advanced Security rules should you enable on Server2?





Explanation:

To use the Computer Management console to manage Server2, ensure that the following firewall rules are enabled:

  • COM+ Network Access (DCOM-In)
  • All rules in the Remote Event Log Management group

These rules allow for remote management and event log access with the principle of least privilege.

Correct Answers:

A. the COM+ Network Access (DCOM-In) rule

B. all the rules in the Remote Event Log Management group
