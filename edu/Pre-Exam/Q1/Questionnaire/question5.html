Question 5

The Default Domain Policy Group Policy Object (GPO) is shown in the GPO exhibit. (Click the GPO tab.)

The members of a group named Service Accounts are shown in the Group exhibit. (Click the Group tab.)

An organizational unit (OU) named ServiceAccounts is shown in the OU exhibit. (Click the OU tab.)

You create a Password Settings Object (PSO) as shown in the PSO exhibit. (Click the PSO tab.)

For each of the following statements, select Yes if the statement is true. Otherwise, select No.

Explanation:

Yes - because the fine-grained password policy applies to the ServiceAccounts group, of which ServiceAccount1 is a member. Fine-Grained Password Policies override Default Domain Policy GPO password settings.

No - because ServiceAccount2 is not a member of the ServiceAccounts security group, it receives the Default Domain Policy GPO password settings.

No - because the minimum password length for this group is at least 16 characters.
