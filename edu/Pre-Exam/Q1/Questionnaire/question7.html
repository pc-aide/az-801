Question 7

You have an Azure virtual machine named VM1 that runs Windows Server.

You plan to deploy a new line-of-business (LOB) application to VM1.

You need to ensure that the application can create child processes.

What should you configure on VM1?





Explanation:

To ensure that a line-of-business (LOB) application can create child processes on a Windows Server VM, you should configure Exploit Protection. Exploit Protection is a feature that helps protect your applications and the system from exploit attempts.

Correct Answer:

D. Exploit protection
