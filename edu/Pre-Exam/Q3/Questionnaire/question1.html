Question 1

Your network contains an Active Directory Domain Services (AD DS) domain. The domain contains a failover cluster named Cluster1.

You need to configure Cluster-Aware Updating (CAU) on the cluster by using Windows Admin Center (WAC).

Which three actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.

Actions:

Add a WAC gateway
Add Cluster1 to WAC
Add a group managed service account (gMSA)
Enable CredSSP
Add the Cluster-Aware Updating role

Drop Zones:

Explanation:

To configure Cluster-Aware Updating (CAU) on the cluster using Windows Admin Center (WAC), follow these steps:

  • Add Cluster1 to WAC
  • Enable CredSSP
  • Add the Cluster-Aware Updating role

Correct Answers:

  • Add Cluster1 to WAC
  • Enable CredSSP
  • Add the Cluster-Aware Updating role
