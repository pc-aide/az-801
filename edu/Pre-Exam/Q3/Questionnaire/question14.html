Question 14

You have three servers named Host1, Host2, and VM1 that run Windows Server. Host1 and Host2 have the Hyper-V server role installed. VM1 is a virtual machine hosted on Host1. You configure VM1 to replicate to Host2 by using Hyper-V Replica.

Which types of failovers can you perform on VM1 on each host? To answer, select the appropriate options in the answer area.

Correct Answers:
VM1 on Host1: Planned Failover only
VM1 on Host2: Test Failover & Failover only

Explanation

Where is Failover initiated?

  • Test - On the replica virtual machine
  • Planned - Initiated on primary and completed on secondary
  • Unplanned - On the replica virtual machine
