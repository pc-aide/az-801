Question 18

You have an on-premises server named Server1 that runs Windows Server and has the Hyper-V server role installed.

You have an Azure subscription.

You plan to back up Server1 to Azure by using Azure Backup.

Which two Azure Backup options require you to deploy Microsoft Azure Backup Server (MABS)? Each correct answer presents a complete solution.

NOTE: Each correct selection is worth one point.




Correct Answers:
A. Bare Metal Recovery
D. Hyper-V Virtual Machines

Explanation

For Hyper-V virtual machines, you need MABS. To back up Server1 to Azure using Azure Backup, the options that require you to deploy Microsoft Azure Backup Server (MABS) involve more complex backup scenarios and require MABS to facilitate the backup process. MABS acts as a bridge between your on-premises servers and Azure Backup, enabling features such as bare-metal recovery and backup of Hyper-V virtual machines.
